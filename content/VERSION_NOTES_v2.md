# SEO and Copy-Edit Pass — v2

This package adds a final technical SEO pass to the live GitHub Pages site.

Changes included:
- Page-specific title tags and meta descriptions
- Canonical URL tags for every public page
- Absolute Open Graph and Twitter image URLs
- Organization, WebSite, and WebPage JSON-LD structured data
- Updated XML sitemap with lastmod values
- Verified robots.txt points to the production sitemap
- Minor copy consistency review; no strategic positioning changes

Deploy by copying these files over the existing repository files, committing, and pushing to GitHub.
